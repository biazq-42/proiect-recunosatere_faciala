# Proiectce are ca scop Recunoașterea Facială

Acesta este un proiect care folosește Python pentru a recunoaște fețe în timp real folosind camera web.

## Cum funcționează?
Programul citește pozele din folderul `imagini_cunoscute`, învață trăsăturile feței, și apoi deschide camera web pentru a vedea dacă persoana din fața camerei este aceași cu cea din poză.

## Instalare

1. Clonează acest repo.
2. Asigură-te că ai Python instalat.
3. Instalează bibliotecile necesare:
   ```bash
   pip install -r requirements.txt
